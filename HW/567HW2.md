# Dead Code

Dead code คือโค้ดที่ถูก Execute แต่ไม่ได้ส่งผลกระทบอะไรเลยกับผลลัพธ์ หรือ ผลลัพธ์ที่ได้จากโค้ดส่วนนั้นไม่ถูกใช้งาน บางแหล่งก็บอกว่าโค้ดที่ไม่ถูก execute ก็เป็น Dead code แต่บางแหล่งก็เรียกว่า unreachable code Deadcode อาจจะส่งผลต่อประสิทธิภาพของโปรแกรมได้

## Example 1: ตัวแปรที่ไม่ได้ใช้

```c
void example2() {
    int x = 5;
    int y = 10;
    printf("%d", y);
}
```

จะเห็นว่า ตัวแปร x นั้นไม่ได้ถูกเรียกใช้งานควรจะลบออก

```c
void example2() {
    int y = 10;
    printf("%d", y);
}
```

## Example 2: Unreachable Code

```c
void example1() {
    int x = 10;
    return;
    x = 20;
}
```

โปรแกรมจบลงใน Line ที่ 3 ทำให้ Line ที่ 4 ไม่ถูก Execute

```c
void example1() {
    int x = 10;
    x = 20;
    return;
}
```

##กรณีที่Dead code ไม่ได้มีปัญหาคือ
โค้ดที่ไม่ได้ใช้งานและถูก comment เอาไว้ อาจจะเป็นโค้ดที่มีบัค ฟีเจอร์ที่ไม่ได้ใช้งานแล้ว หรือโครงสร้างโค้ดของระบบในอนาคต
เช่น

```c
void process() {
    int result = calculate();
    // printf("Debug result: %d", result);
    return;
}
```
